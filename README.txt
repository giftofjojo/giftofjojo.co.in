Gift Of JoJo Live Media Website

Files:
- index.html: Main website content, videos, contact details, and event cards.
- styles.css: Website design, responsive layout, dark/light theme, and animations.
- script.js: Mobile menu, theme toggle, sticky header, and copy-link behavior.
- gift-of-jojo-logo.png: Channel logo used across the website.

Editing videos:
Search index.html for "youtube.com/embed/" and replace the video IDs with new YouTube video IDs.

Editing contact details:
Search index.html for "contact-list" and update the email or phone links.

Editing upcoming live events:
Search index.html for "events-section" and update the TBA cards with real event dates, titles, and descriptions.

Preview:
Open index.html in a browser, or run a local server from this folder.
